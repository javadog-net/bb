# 1.提供ECU-获取经销商列表接口

# 1.1 地址

baseUrl + /api/euc/dealers

# 1.2 请求方式

POST

# 1.3 参数

参数 注释 是否必填 备注
companyCode 经销商编码 否 如：8700099146 (模糊搜索)
companyName 经销商名称 否 如：邯郸市阳光三联电器有限公司 (模糊搜索)
gmId 中心编码 否 如：12A02
pageNo 页数 否 -
pageSize 每页显示数量 否 -

# 1.4 返回值
